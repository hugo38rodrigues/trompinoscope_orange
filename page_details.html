Affiche l'aide sur les recherches

Lors d'une recherche via le « champ multicritère » :
  • L'Annuaire effectue par défaut une recherche APPROXIMATIVE, mais si vous le souhaitez, vous pouvez sélectionner manuellement le mode de recherche PHONÉTIQUE.
  • Le mode de recherche par défaut est configurable via le menu « Mon annuaire », « Mes paramètres »
  • Vous devez écrire entièrement votre mot ou groupe de de mots avec l'orthographe qui vous semble la plus juste.
  • Exemples de recherches :
    • dupont paris
    • paris dupont
    • marie dupont

Remarques :
La recherche APPROXIMATIVE
Permet de trouver des résultats même si la requête comporte des erreurs ou des variations mineures (fautes de frappe, synonymes, ...).

La recherche PHONÉTIQUE
Se base sur la prononciation des mots pour identifier des correspondances, même si l'orthographe diffère.

Affiche l'aide sur les recherches

Lors de la recherche sur la fiche personne par "nom-prénom" ou en mode "avancée", il est possible de reproduire les options "égal à", "commence par", "contient" et "finit par" en utilisant simplement la liste déroulante associée aux champs :

  • En cas de doute sur l'orthographe exacte du nom et/ou prénom, choisissez "commence par", "finit par", ou "contient"
    • Exemple Nom : dupon (option "commence par") présentera toutes les personnes dont le nom commence par dupont, dupond, etc...
  • Recherche phonétique (1): il suffit de cocher la case associée.
  • Localisation: en cas de recherche sur la ville, le site, le code postal ou département, vous pouvez ne pas obtenir de résultat, si aucune adresse de visite n'est renseignée dans la fiche du collaborateur.

Remarque:
(1) La phonétique est une branche de la linguistique qui étudie les sons utilisés dans la communication verbale.

Les recherches via "nom-prénom" et "avancée" utilisent un système de transcription phonétique pour représenter les sons du langage.

Affiche l'aide sur les recherches

Lors d'une "recherche" sur une entité, vous pouvez utiliser son sigle, son chemin hiérarchique ou des mots clés de la description (responsable, ville, code postal ou département) :

  • En cas de recherche sur la ville, le code postal ou le département, seules les entités ayant une adresse de visite seront incluses dans les résultats.
  • En cas de doute sur l'orthographe, il est possible d'utiliser le caractère * qui remplace un nombre quelconque de caractères pour compléter votre recherche :
    • Par exemple sur le sigle : DTSI ou *TSI ou DTS* ou *DTS*
    • Par exemple sur le chemin hiérarchique :
      • DRH / RH
      • *DRH* / RH
      • DTSI / * / RH
    • Par exemple sur la ville :
      • Paris
      • Pari*

Que puis-je demander au CAR ?


Les données d'identités et de rattachement hiérarchique sont référencées dans le GIR (Group identity Referential / Référentiel d'Identité Groupe).


En ce qui concerne les fiches annuaire du périmètre France, les modifications portant sur ces données (civilité, nom, prénom et Responsable/Manager hiérarchique et visibilité), il est nécessaire de faire une demande par l'outil DMI.


L'Annuaire Groupe est le référentiel des données de contact et du rattachement organisationnel/entités. Les modifications sont opérées par le Correspondant Annuaire de l'entité d'appartenance (CAR).


A noter que le titulaire d'une fiche annuaire dispose de la capacité de mettre à jour les informations de l'onglet fonction de sa propre fiche (fonctions activités, missions, expertises) et il n'est pas nécessaire de faire de demande au CAR.


Ainsi, la complétude et l'exactitude des informations sur votre fiche annuaire est de votre responsabilité. Dès lors, plus votre fiche annuaire est renseignée et riche en informations, il serait plus aisé pour les collègues de vous retrouver durant leur recherche.
